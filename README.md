Git Coreutils
=============

Coreutils for Git repository. They work on bare repository.

Installtion
-----------

Put them all into any directory in PATH. e.g.) /usr/bin

Usages
------

#### git-cp

    $ git cp <src> <repo.git>
    $ git cp <src> <repo.git>

#### git-cat

    $ cd <repo.git>
    $ git cat <filename>
